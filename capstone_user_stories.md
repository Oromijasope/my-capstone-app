## Exercise 3: Login and Registration
* **Registration:** As a new user, I want to register by entering my username, email, and password, and clicking the Sign Up button so that I can create an account.
* **Login:** As a returning user, I want to input my email and password and click the Login button so that I can access my profile.
* **Error Feedback:** As a user, I want to receive a message if I leave fields empty or enter incorrect details so that I know what went wrong.
* **Store Data:** As a user, I want my details saved in local storage so that I don't have to log in every single time.

## Exercise 4: Home Screen
* **Data Overview:** As a user, I want to view an overview of my data on the home screen so that I can monitor my progress at a glance.
* **Onboarding:** As a new user, I want to see a quick introductory guide on the home screen so that I can learn how to use the app.
* **Efficient Navigation:** As a user, I want to access my most-used features from the home screen so that I can navigate the app efficiently.
